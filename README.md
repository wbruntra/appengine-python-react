This is a simple starter project for Python App Engine Standard.

### Frameworks Used:
- Flask-Restful (Backend)
- React (Frontend)

### Technology
- Python App Engine (Standard, using Python 2.7)
- webpack
- webpack-dev-server (For development)

## Getting Started

### `client` directory

1. `yarn install`

### main directory

1. `npm run setup`
2. `npm run dev`
3. Dev server at `http://localhost:3000`
